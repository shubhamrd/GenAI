# 🧠 GenAI Documents Repository

A comprehensive collection of Generative AI documentation, demos, and learning resources focused on AWS BedRock and related technologies.

```
GenAI/
├── README.md                          # This file
├── AWS BedRock Docs/                  # AWS BedRock documentation and demos
│   ├── genai-demo.html               # HTML-based GenAI demo

└── ...                                # Additional GenAI materials
```

## 📖 Contents

This repository contains materials for learning and practicing Generative AI concepts:

### AWS BedRock Documentation & Demos
- **genai-demo.html** - Interactive web-based GenAI demonstrations
- **genai-lambda-demo.md** - Serverless GenAI implementations using AWS Lambda
- **module2-demo.ipynb** - Hands-on Jupyter notebook demos for GenAI concepts

### Topics Covered
- Large Language Models (LLMs)
- AWS BedRock services and API integration
- Prompt engineering techniques
- RAG (Retrieval-Augmented Generation)
- GenAI application development
- Serverless GenAI implementations

## 🚀 Getting Started

1. Browse the documentation folders to explore available resources
2. Review the Jupyter notebooks for hands-on learning
3. Try the web-based demos in your browser
4. Read the Lambda demo for serverless GenAI patterns
